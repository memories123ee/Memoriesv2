           📔 كيف تشارك ذكرياتك - دليل سريع

┌─────────────────────────────────────────────────────────┐
│  1️⃣ لمشاركة ذكرياتك مع شخص آخر (ليراها)              │
└─────────────────────────────────────────────────────────┘

  خطوات بسيطة:
  
  أنت:
  ✅ اضغط زر التنزيل ⬇️ (تصدير المذكرات)
  ✅ سيتم تحميل ملف JSON
  ✅ أرسل هذا الملف + رابط الموقع للشخص
  
  الشخص الآخر:
  ✅ يفتح رابط الموقع
  ✅ يضغط زر الرفع ⬆️ (استيراد المذكرات)
  ✅ يختار ملف JSON الذي أرسلته
  ✅ يرى جميع ذكرياتك! 🎉

  مثال:
  واتساب → أرسل الملف JSON
  رسالة: "افتح هذا الرابط واضغط زر ⬆️ واختر الملف"
  ✅ تم!


┌─────────────────────────────────────────────────────────┐
│  2️⃣ لإعطاء الموقع لشخص (ليصنع ذكرياته الخاصة)       │
└─────────────────────────────────────────────────────────┘

  أسهل طريقة:
  
  ✅ فقط أعطه رابط الموقع
     مثال: https://username.github.io/memory-book
  
  ✅ هو يفتح الرابط
  
  ✅ سيرى موقع فارغ (هذا طبيعي!)
  
  ✅ يبدأ يصنع ذكرياته الخاصة
  
  ✅ ذكرياته لن تختلط مع ذكرياتك أبداً!
  
  ملاحظة مهمة:
  ⚠️ كل شخص له بياناته الخاصة في متصفحه
  ⚠️ لن يرى ذكرياتك ما لم ترسل له ملف JSON


┌─────────────────────────────────────────────────────────┐
│  3️⃣ الفرق بين الطريقتين                               │
└─────────────────────────────────────────────────────────┘

  الطريقة 1 (مشاركة ذكرياتك):
  ═══════════════════════════════════
  أنت → تصدير → إرسال ملف JSON → استيراد
  النتيجة: يرى ذكرياتك ✅
  
  الطريقة 2 (موقع فارغ):
  ═══════════════════════════════════
  أنت → إرسال رابط فقط
  النتيجة: يرى موقع فارغ ويصنع ذكرياته ✅


┌─────────────────────────────────────────────────────────┐
│  📍 أمثلة عملية                                        │
└─────────────────────────────────────────────────────────┘

  مثال 1: هدية لزوجتك 💝
  ═══════════════════════════
  1. تصنع دفتر ذكريات جميل
  2. تصدّر (زر ⬇️)
  3. ترسل الملف + الرابط لها
  4. هي تستورد (زر ⬆️)
  5. ✅ مفاجأة رائعة!

  مثال 2: صديق يريد استخدام الموقع 🤝
  ═══════════════════════════════════════
  1. تعطيه الرابط فقط
  2. هو يفتحه
  3. يرى موقع فارغ
  4. يبدأ يصنع ذكرياته
  5. ✅ لن يرى ذكرياتك!


┌─────────────────────────────────────────────────────────┐
│  🔒 الأمان                                              │
└─────────────────────────────────────────────────────────┘

  ✅ الموقع = عام (أي شخص يفتحه)
  ✅ البيانات = خاصة (في متصفح كل شخص)
  
  مثال:
  - أنت تفتح الرابط → ترى ذكرياتك
  - صديقك يفتح نفس الرابط → يرى موقع فارغ
  
  السبب:
  البيانات محفوظة في متصفحك فقط، وليس على الإنترنت!


┌─────────────────────────────────────────────────────────┐
│  ⚠️ نصيحة مهمة                                         │
└─────────────────────────────────────────────────────────┘

  صدّر نسخة احتياطية كل أسبوع!
  
  ✅ اضغط زر ⬇️
  ✅ احفظ الملف في:
     - Google Drive
     - البريد الإلكتروني
     - فلاش USB
  
  لماذا؟
  إذا مسحت بيانات المتصفح = ضاعت الذكريات!

  1. افتح هذا الرابط: [رابط الموقع]
  2. اضغط على زر السهم للأعلى ⬆️ في الأعلى
  3. اختر الملف الذي أرسلته لك
  4. ستشاهد المذكرات!
